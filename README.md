# Auto News Website (Clean Version)

This is a cleaned version of the Auto News Website project, safe for public sharing on GitHub. All sensitive API keys and credentials have been removed.

## Setup
1. Copy `config.example.json` to `config.json` and fill in your own API keys and settings.
2. Copy `client_secret.example.json` to `client_secret.json` and add your Google OAuth credentials.
3. Install dependencies from `requirements.txt`.
4. Follow the setup guides in the markdown files for Blogger and Facebook.

## Sensitive Files
- `config.json` and `client_secret.json` are excluded via `.gitignore`.
- Never commit your real API keys or credentials.

## Usage
- See the main scripts: `auto_post_1min.py`, `auto_post_dual_platform.py`, and `main.py`.
- Modules are in the `modules/` directory.
